Detailed Description

cFrequencyCounter is a basic frequency counter. Frequency is defined as the number of events of a particular sort occurring in a set period of time. The period of time is adjustable but is set at 1 second as a default value.
By calling the method signal() a counter is incremented. When the internal clock reaches the programmed timeout period (1 second for instance), the frequency value is computed and updated.
