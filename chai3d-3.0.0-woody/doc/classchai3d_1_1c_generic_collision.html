Detailed Description

A collision detection algorithm is always created for a specific object (mesh) passed by reference in the constructor. The collision detector must then be initialized by calling method initialize() which computes and builds the necessary data structures for the particular object.
The initialize() method takes one argument named a_radius which defines a boundary distance around every triangle (an enclosing shell). When the collision inquiry method is called computeCollision(), collisions are searched between a segment, passed as argument, and the shells of every triangle. If this radius is set to zero, then the shells are equal to the triangles. This option is used by the finger-proxy force rendering algorithm when compute the intersection between a sphere (contact point) and the surface of a mesh.
If the shape of the object is modified, or if triangles are added or removed for instance, then the initialize() command of the collision detector must be called again. The method is responsible for deallocating any previously built data structures.
Please note that this class does not support collision detection between objects themselves.
